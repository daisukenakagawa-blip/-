# サブアカウント（2つ目のYouTube）への自動投稿 設定手順

メインと同じ仕組みで、2つ目のYouTubeアカウントにも自動投稿できます。
**やることは「サブ用トークンを1回だけ作って GitHub に登録する」だけ**です。

---

## 事前の注意（重要）

- サブアカウントのトークンは、**メインと同じ `client_secret.json`（同じ Google Cloud プロジェクト）** で作ってください。
  そうすれば GitHub には `YOUTUBE_TOKEN_2` を登録するだけで済みます。
- Google Cloud の「OAuth 同意画面」が **「テスト」モード** の場合、
  2つ目のアカウントのメールアドレスを **テストユーザー** に追加しておいてください。
  （Google Cloud Console → API とサービス → OAuth 同意画面 → テストユーザー → 追加）
  これをしないと認証時に「アクセスをブロックしました」と出ます。

---

## STEP 1. サブ用トークンを作る（自分のPCで1回だけ）

メインのトークンを作ったときと同じPC（ブラウザが使える環境）で行います。

1. プロジェクトの `auto_video_uploader` フォルダを開く
2. **いまある `token.json` を別名に退避**（メインのを消さないため）
   ```bash
   mv token.json token_main.json
   ```
3. 認証コマンドを実行
   ```bash
   python main.py --auth-only
   ```
4. ブラウザが自動で開くので、**必ず “2つ目（サブ）のYouTubeアカウント” でログイン**して「許可」
   - ※ふだんのGoogleにログイン済みだとメインが選ばれがちなので、アカウント選択画面で**サブを選ぶ**こと
5. 認証が終わると、新しい `token.json`（＝サブ用）が作られます
6. 中身をあとで使うので、このファイルを開いておく
   - 退避したメイン用に戻したい場合は、サブの token.json を `token_sub.json` にリネームしてから
     `mv token_main.json token.json` で元に戻せます

---

## STEP 2. GitHub にサブ用トークンを登録する

1. GitHub のリポジトリページを開く
2. **Settings**（設定）→ 左メニュー **Secrets and variables** → **Actions**
3. **New repository secret**（新しいシークレット）をクリック
4. 次のように入力：
   - **Name**：`YOUTUBE_TOKEN_2`
   - **Secret**：STEP1で作った **サブ用 token.json の中身を丸ごと貼り付け**
     （`{` から `}` まで全部）
5. **Add secret** で保存

> 同じ Google Cloud プロジェクトを使うなら `YOUTUBE_CLIENT_SECRET_2` は不要です
> （メインの `YOUTUBE_CLIENT_SECRET` が自動で使われます）。

---

## STEP 3. サブアカウントに投稿する

GitHub の画面から手動で実行します。

1. リポジトリの **Actions** タブ → 左の **voiceover-publish** を選ぶ
2. 右上 **Run workflow** をクリック
3. 入力欄を設定：
   - **story**：投稿したい素材フォルダ名（例：`voiceover_story_sub1`）
   - **account**：`2`  ← これでサブアカウントに投稿
   - **publish_at_jst**：予約時刻（例：`2026-07-01 18:00`／空なら即時公開）
4. **Run workflow** で実行 → 数分後にサブアカウントへ投稿されます

---

## まとめ

| ステップ | 誰がやる | 頻度 |
|---|---|---|
| サブ用トークン作成（STEP1） | あなた（PCでブラウザログイン） | 最初の1回だけ |
| `YOUTUBE_TOKEN_2` 登録（STEP2） | あなた（GitHub操作） | 最初の1回だけ |
| 別内容の動画素材づくり | Claude（私）が作成 | 毎回 |
| サブへ投稿（STEP3） | 手動 or 私がトリガー | 毎回 |

トークン登録（STEP1・2）さえ終われば、あとは
**「サブ用にこういう動画を作って account=2 で出して」** と言うだけで、
メインと別内容・別時刻で自動投稿できます。
