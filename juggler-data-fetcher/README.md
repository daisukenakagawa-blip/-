# ジャグラー台データ取り込みツール（juggler-data-fetcher）

ホールデータサイトの台データ（総回転数・BB・RB・差枚）を、予想ツール
（`../juggler-yosou.html` など）で使える CSV/JSON に変換するための
**サーバー/ローカル実行用スクリプト**です。

---

## ⚠️ 最初に必ず読んでください（規約・法的注意）

- **多くのデータサイトは利用規約でスクレイピング/自動収集を禁止しています。**
  実行する前に、対象サイトの**利用規約と `robots.txt` を自分で確認**し、
  許可されている範囲でのみ使ってください。違反は規約違反・アクセス遮断・
  法的トラブルの原因になります。
- **ログイン必須・有料会員・スクレイピング対策のあるサイト
  （例：データロボ サイトセブン）は対象外です。** アクセス制御の回避は
  行いません。本ツールもそうした突破機能は持ちません。
- 取得は**個人利用の範囲**にとどめ、サーバーに負荷をかけないよう
  **レート制限（既定3秒間隔）**を守ってください。再配布・商用利用は不可。
- このツールは `robots.txt` を自動取得し、対象パスが `Disallow` の場合は
  **取得を拒否**します。

> このリポジトリの実行環境（クラウド）は外部アクセスが許可制のため、
> 多くのデータサイトに到達できません。**`url` モードはあなたのPC/サーバーで
> 実行**してください。ネットに出られない環境では、安全な **`file` モード
> （保存したHTMLを解析）** を使えます。

---

## 必要環境
- Node.js 18 以上（標準モジュールのみ。追加インストール不要）

## 使い方

### 1) まず構造を確認（inspect）
ページ内の表を一覧し、各列の中身サンプルを表示します。どの列が
台番号/G数/BB/RB/差枚かを確認するために使います。

```bash
# URLから（自分の環境で・規約を確認のうえ）
node juggler-fetch.js inspect "https://example-hall-data/▲▲"

# 保存したHTMLから（ネット不要・安全）
node juggler-fetch.js inspect ./saved.html
```

### 2) CSVに変換（url または file）
`inspect` で分かった列番号を `--cols` で指定します（0始まり、
順番は 台番号,G数,BB,RB,差枚,機種）。差枚・機種など不要な列は `-` で
飛ばせます。**機種列を入れておくと、予想ツール側で機種が自動選択**されます。

```bash
# 例: 表の0列=台番号, 1列=機種, 2列=G数, 3列=BB, 4列=RB, 6列=差枚
node juggler-fetch.js file ./saved.html --table 0 --cols "0,2,3,4,6,1" --out daily.csv

# URLモード（robots.txtを尊重・レート制限あり）
node juggler-fetch.js url "https://.../hall?date=20260619" \
  --table 1 --cols "0,2,3,4,6,1" --delay 4000 \
  --ua "my-personal-juggler-tool (contact: you@example.com)" --out daily.csv
```

出力 `daily.csv`:
```
台番号,機種,総回転数,BIG,REG,差枚,合成確率
101,マイジャグラーV,5210,21,18,1840,1/133.6
...
```

### 3) ラベル方式（単台ページ向け）
表ではなく「総スタート 5210 / BB 21 / RB 18 / 差枚 +1840」のような
単台ページは `--label` で見出し語の近くの数字を拾います。

```bash
node juggler-fetch.js file ./single.html --label --out one.csv
```

## 予想ツールへの取り込み（一気通貫）
出力した `daily.csv` を `juggler-yosou.html` の上部
**「📥 CSVから一括取り込み」** で読み込むと、台一覧から台を選ぶだけで
G数・BB・REG・差枚（と機種）が自動入力され、そのまま設定推定＆展開予想が
走ります。手入力・写真OCR不要の一気通貫フローです。
