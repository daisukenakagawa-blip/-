# ♠ ビデオポーカー (Jacks or Better)

スマホで遊べる HTML5 ポーカーゲーム。PWA 対応でホーム画面に追加すれば「アプリ」として起動でき、オフラインでも遊べます。

## 遊び方
1. `mobile-game/index.html` をブラウザで開く（スマホ推奨）
2. ＋／－／MAX でベットを決めて **DEAL**
3. 残したいカードをタップして **HOLD**
4. **DRAW** で残りを引き直し、役が成立すると配当ゲット

ジャック以上のペアで配当成立。クレジットはブラウザに自動保存されます。

## 配当表（ベット倍率）
| 役 | 倍率 |
|---|---|
| ロイヤルフラッシュ | ×250 |
| ストレートフラッシュ | ×50 |
| フォーカード | ×25 |
| フルハウス | ×9 |
| フラッシュ | ×6 |
| ストレート | ×4 |
| スリーカード | ×3 |
| ツーペア | ×2 |
| ジャック以上のペア | ×1 |

## スマホアプリとして使う
1. スマホのブラウザで `index.html` を開く
2. ブラウザメニューから「ホーム画面に追加」
3. アイコンから全画面・オフラインで起動

## ローカルで試す
PWA（Service Worker）を有効にするには `http://` 経由でのアクセスが必要です:
```bash
cd mobile-game
python3 -m http.server 8000
# http://localhost:8000 を開く
```

## 構成
- `index.html` — ゲーム本体（HTML/CSS/JS、依存なし）
- `manifest.json` — PWA マニフェスト
- `service-worker.js` — オフラインキャッシュ
- `icons/` — アプリアイコン
