# note 勝てるニッチ発掘ツール（note_niche）

「いい記事を書く」前に、**売れる場所（土俵）を選ぶ**ためのツールです。

副業・NISA・ChatGPT のような**飽和テーマ**は、実績ある大型アカウントに支配されていて、
無名が一般論を書いても埋もれます。このツールは逆を行きます——
**「需要はあるのに、強い有料記事が少ない“すき間”」**を、note の実データで見つけます。

## 何をするか

1. あなたの興味（seeds.csv）から、**ロングテールのニッチ候補**を量産（AI、任意）
2. 各候補について **note の公開検索で上位記事を実取得**
3. 次の2軸で**スコア化**：
   - **需要**：上位記事の反応（スキ数）＝人々が関心を持っているか
   - **競合**：強い有料記事が何本あるか／大型アカウントが支配していないか
4. **「需要◎ × 有料競合が弱い」キーワードだけ**を「狙い目」として抽出・ランキング
5. 任意で、狙い目ニッチに **AIが攻略案（切り口・タイトル・価格）** を付ける

> スコア = 需要(0〜50) − 有料競合(0〜60) + 具体性ボーナス(0〜10)。全て透明な計算です。

## なぜこれが効くか（前回までの反省）

- 「副業ロードマップ」は何千件もある＝**コモディティ**。文章を磨いても売れない。
- noteは「誰が・どんな実績で」で売れる。無名が勝てるのは、**大手がまだ書いていない具体的なすき間**。
- このツールは、その**すき間をデータで先に特定**します。書くのはその後。

## 使い方

### Windows
| ファイル | やること |
|---|---|
| `①セットアップ.bat` | 部品の導入 |
| `②ニッチを探す.bat` | seeds.csv の興味から狙い目ニッチを探索 |

### コマンド
```bash
pip install -r requirements.txt
python find.py                              # seeds.csv → 候補展開 → 採点
python find.py "経理 在宅 副業 簿記2級" "保育士 復職 不安"   # 候補を直接採点
```

候補の渡し方は3通り（上から優先）：**引数** → `candidates.csv`（手動リスト）→ `seeds.csv`＋AI生成。

## 出力（`output/`）

- `niche_日時.md` … 🎯狙い目ニッチのランキング＋（AI）攻略案
- `niche_日時.csv` … 全キーワードのスコア（Excelで分析可）

スコアの読み方の例：

| 判定 | 意味 |
|---|---|
| 🎯 狙い目 | 需要があり、有料競合が弱い。**ここを書く。** |
| 条件付き | 需要はあるが競合も拮抗。切り口で差別化できれば可。 |
| 回避 | レッドオーシャン（強い有料＋大型アカウント）。 |
| 需要薄(要検証) | 記事が少なく需要が小さい可能性。 |

## 設定（`config.py` の調整つまみ）

| 項目 | 説明 | 既定 |
|---|---|---|
| `WEAK_PAID_LIKES` | 最強有料記事がこのスキ未満なら「競合が弱い＝穴」 | 50 |
| `BIG_AUTHOR_FOLLOWERS` | 上位にこの規模の著者がいたら減点 | 10000 |
| `SCORE_GOOD` / `SCORE_MAYBE` | 「狙い目」「条件付き」の閾値 | 25 / 10 |
| `MIN_NOTES_FOR_DEMAND` | これ未満の記事数は「需要薄」 | 8 |
| `CANDIDATES_PER_SEED` | 1シードから作る候補数（AI使用時） | 25 |

## 全体の流れ（他ツールとの連携）

```
note_niche（土俵選び）→ 狙い目ニッチを決定
   → note_factory（編集部AIで記事化）
   → note_publisher（下書き投稿）
```

## 精度を上げるコツ

- `seeds.csv` には、**あなたが少しでも経験/知識のある分野**を入れる（最終的に書けるニッチが出る）。
- 「狙い目」が出たら、その**実際の上位記事を自分の目で確認**（数字は入口、最後は人間の判断）。
- noteの公開検索を控えめな間隔（既定1秒）で利用します。短時間の大量アクセスは避けてください。

## 注意

- note は非公式の公開エンドポイントを使うため、仕様変更で取得できなくなる場合があります。
- スコアはあくまで「土俵選びの補助線」です。最終判断はあなたが行ってください。
