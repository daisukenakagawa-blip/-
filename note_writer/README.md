# note 売れる記事ジェネレーター(量産エンジン)

`note_research`(第1ステップ)で調べた売れ筋をもとに、**note で売れる構成の記事を
量産する**ツールです。自動記事量産ツールの**第2ステップ(執筆)**にあたります。

## 構成

```
note_writer/
├─ articles/            ← 完成サンプル記事 10本(そのまま販売できる下書き)
├─ generator.py         ← 記事を量産するエンジン(これを実行)
├─ article_plan.csv     ← 量産するテーマ一覧
├─ config.py            ← 設定
├─ output/              ← 自動生成した記事の出力先
└─ README.md
```

## 量産する(`generator.py`)

```bash
pip install -r requirements.txt          # note_research と同じ requirements でOK
python generator.py                       # article_plan.csv の全テーマを生成
python generator.py "副業 在宅 始め方"      # テーマ直接指定で1本だけ
```

- `.env` に `ANTHROPIC_API_KEY` を設定すると **AI が本文を執筆**します
- 未設定でも、売れる構成が埋まった **雛形**(書き足し用)を出力します
- `note_research/output` に最新のリサーチCSVがあれば、**価格中央値**を自動反映します

生成物は `output/` に Markdown で保存されます。

## LINE通知の設定(記事ができたらLINEに通知)

`.env` に LINE のトークンを設定すると、**記事の生成が終わったタイミングで
自分のLINEに通知**が届きます(未設定なら自動でスキップ)。

> ※ 以前の「LINE Notify」は2025年3月末で終了したため、現在は **LINE Messaging API**
> を使います。無料で使えます。

### 手順(最初に1回・約5分)

1. [LINE Developers](https://developers.line.biz/) にLINEアカウントでログイン
2. 「新規プロバイダー」を作成 →「**Messaging API**」のチャネルを新規作成
   (これがあなた専用の通知用「公式アカウント」になります)
3. チャネルの「**Messaging API設定**」タブで:
   - 一番下の「**チャネルアクセストークン(長期)**」を発行してコピー
   - 表示されたQRコードから、**自分のLINEでその公式アカウントを友だち追加**
4. `note_writer/.env` に貼り付け:
   ```
   LINE_CHANNEL_ACCESS_TOKEN=（コピーしたトークン）
   ```
5. これで完了。`python generator.py` を実行すると、生成完了時にLINEへ通知が届きます。

> 自分だけに送る用途なら **userID の取得は不要**です(友だち全員＝自分へ broadcast)。
> 友だちが複数いて特定の人だけに送りたい場合のみ `LINE_USER_ID` を設定してください。

## 売れるためのコツ(重要)

1. **タイトルが9割**。数字・【】・ベネフィットを入れる(例「【2026年版】〜3つの方法」)
2. **無料パートで出し惜しみしない**。価値を見せて「続きが読みたい」を作る
3. **テンプレ・チェックリスト**など"持ち帰れるもの"を必ず入れる
4. **誇大表現・虚偽実績はNG**(規約違反・景表法/薬機法リスク)。ノウハウで売る
5. 価格は**¥500前後から**始め、反応を見て調整する

## 法令・規約の注意

- 投資・お金・健康のテーマは、利益や効果を**保証する断定表現を避け**、
  「情報提供/自己責任/専門家に相談」の注記を入れてください。
- note の利用規約・各種法令(景品表示法・薬機法など)を遵守してください。
