# ChatGPTを"使える人"になる7つのプロンプト術【保存版】

「ChatGPTに聞いても、当たり障りのない答えしか返ってこない」——
それは、ChatGPTが悪いのではなく**聞き方（プロンプト）**に伸びしろがあるサインです。

このnoteは、すぐ真似できる**7つのプロンプト術**で、同じChatGPTから
「使えるアウトプット」を引き出す方法をまとめました。

## このnoteで分かること

- 凡庸な回答を激変させる7つの型
- そのままコピペで使えるテンプレ
- やりがちなNGな聞き方
- 回答が浅いときの「掘り下げフレーズ」

## なぜ回答が"当たり障りない"のか

ChatGPTは、曖昧な質問には**平均的で無難な答え**を返します。
逆に、**役割・前提・形式**を指定すると、出力は一気に具体的になります。

## 1. 役割を与える（ロールプレイ）

```
あなたはプロのコピーライターです。以下の商品のキャッチコピーを5案、
ターゲット別に作ってください。
```

## 2. 前提条件を渡す

```
前提：読者は初心者／目的は申込み増／トーンは親しみやすく
この前提で、〇〇の紹介文を書いてください。
```

## 3. 出力形式を指定する

```
表形式で。列は「項目／メリット／注意点」。各項目は1行で簡潔に。
```

## 4. 段階的に考えさせる

```
いきなり結論ではなく、まず選択肢を3つ挙げ、それぞれの長所短所を比較し、
最後におすすめを理由つきで述べてください。
```

## 5. 例を見せる（お手本指定）

```
次の例と同じトーン・構成で書いてください。
例：（ここにお手本を貼る）
```

## 6. 制約をかける

```
小学生にも分かる言葉で／専門用語は使わない／300字以内で。
```

## 7. 自己添削させる

```
今の回答を、あなた自身でレビューし、改善版をもう一度出してください。
特に「分かりにくい箇所」を直して。
```

## 回答が浅いときの掘り下げフレーズ集

- 「具体例を3つ追加して」
- 「逆の立場の意見も入れて」
- 「数字やデータの観点を加えて」
- 「実際の手順に落とし込んで」

## やりがちなNGな聞き方

- ❌「〇〇について教えて」（漠然としすぎ）
- ❌ 一度に何個も詰め込む（1プロンプト=1目的が基本）
- ❌ 前提を省いて「いい感じに」と丸投げ

## まとめ

ChatGPTは「役割・前提・形式」を渡すだけで別物になります。
まずは7つの型のうち①役割と③形式を、今日の質問に足してみてください。
