# 後悔しない転職の進め方｜職務経歴書・面接テンプレ付き完全ガイド

「転職したいけど、何から始めればいい?」
「職務経歴書の書き方が分からない」「面接で何を話せば受かるの?」——

このnoteは、勢いで辞めて後悔しないための**転職の正しい順番**と、
そのまま使える**職務経歴書・面接の回答テンプレ**をまとめた実践ガイドです。

## このnoteで分かること

- 辞める前にやるべき「転職の準備順」
- 通る職務経歴書の型（コピペテンプレ）
- 面接の頻出質問と回答フレーム
- 年収交渉の進め方

## 転職で最初にやってはいけないこと

それは**「勢いで先に退職する」**こと。在職中に動くことで、収入を保ちながら
冷静に選べます。まずは辞めずに、情報収集と書類準備から始めます。

## 1. 転職の準備順（在職中にやる）

1. **転職の軸を言語化**（年収？ 働き方？ 仕事内容？ 優先順位をつける）
2. 職務経歴の棚卸し（やってきたこと・成果を箇条書きで全部出す）
3. 転職サイト/エージェントに登録（複数で比較）
4. 職務経歴書・履歴書を作成
5. 応募 → 面接 → 内定 → 退職交渉、の順で進める

## 2. 通る職務経歴書の型

```
■職務要約（3〜4行）
〇〇業界で△年、□□の業務に従事。特に〜の領域で成果を出してきました。

■職務経歴（各社ごと）
・担当業務：
・取り組み（課題→行動→結果の順で）：
・成果（できるだけ具体的に。数字があれば数字で）：

■活かせるスキル
・〇〇／△△／□□

■自己PR（200字程度）
```

> ポイントは**「課題 → 行動 → 結果」**の順で書くこと。「頑張りました」ではなく
> 「何を・どうして・どうなったか」を示します。

## 3. 面接の頻出質問×回答フレーム

**Q. 転職理由は?**
→ ネガティブ（不満）ではなく**前向きな目的**に変換
「現職で〇〇を経験し、次は△△に挑戦したいと考えたため」

**Q. 自己紹介してください**
→ 「①経歴の要約 → ②強み → ③志望との接点」を1分で

**Q. 志望動機は?**
→ 「①なぜこの業界 → ②なぜこの会社 → ③入社後どう貢献できるか」

**Q. 逆質問は?**
→ 「入社後に活躍している人の共通点は?」など**前向きな質問**を準備

## 4. 年収交渉の進め方

- 交渉は基本「**内定後**」に行う
- 「現職の年収」と「希望額」と「その根拠（スキル・実績）」をセットで伝える
- エージェント経由なら、担当者に交渉を代行してもらえる

## 5. 退職をスムーズに進めるコツ

- 退職の意思は**直属の上司に最初に**伝える
- 引き継ぎ計画を自分から提示する
- 立つ鳥跡を濁さず（業界は意外と狭い）

## まとめ

転職は「在職中に・正しい順番で」進めるのが鉄則。
まずは"転職の軸"の言語化と、職務経歴の棚卸しから始めましょう。
