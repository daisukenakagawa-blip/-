# 在宅ワーク未経験から始める「失敗しない仕事選び」完全マニュアル

「在宅で稼ぎたいけど、詐欺っぽい案件ばかりで怖い」
「スキルも資格もない自分にできる仕事があるの?」——

このnoteは、**未経験・スキルなしから始められる在宅ワーク**だけを厳選し、
"危ない案件の見分け方"と"最初の1件の取り方"を手順化したマニュアルです。

## このnoteで分かること

- 未経験OKの在宅ワーク7種と、向き不向きの早見表
- 「初期費用を請求する案件＝危険」など詐欺の見抜き方
- スキマ時間（1日1〜2時間）でも回せる進め方
- 仕事をくれるサービスの登録〜受注テンプレ

## 在宅ワークで最初に知るべき「3つの落とし穴」

1. **高額な初期費用・登録料を要求する案件**は基本すべてNG
2. 「誰でも簡単に高収入」は、ほぼ誇大広告
3. 「マニュアル代」「ツール代」を先払いさせる仕事は避ける

正規の在宅ワークは**働いた分が後から支払われる**のが原則。お金を払う側に
立たされたら、いったん立ち止まりましょう。

## 2. 詐欺案件チェックリスト（保存版）

- [ ] 報酬が相場より極端に高い（1文字10円など）
- [ ] LINEへの誘導後に「教材」を売られる
- [ ] 業務内容が曖昧なまま「まず登録」を急かす
- [ ] 運営会社の所在地・連絡先が確認できない

ひとつでも当てはまったら撤退でOKです。

## 3. スキマ時間で回す「1日の型」

- 朝の15分：応募・連絡の返信だけ
- 昼/夜の60〜90分：実作業（タイマーで区切る）
- 寝る前5分：翌日のタスクを1行メモ

「まとまった時間」を待つと一生始められません。**15分単位**で分割します。

## 4. 受注メッセージのテンプレ

```
ご依頼を拝見しました、〇〇と申します。
未経験ではありますが、丁寧さと納期厳守を強みに対応いたします。

・着手前に、認識合わせのため作業内容を一度確認させてください
・小さな確認もこまめにご連絡します

まずは少量からお任せいただければ幸いです。
```

未経験を隠す必要はありません。**「丁寧・報連相・納期厳守」**を明言する方が
むしろ選ばれます。

## 5. 最初の1ヶ月のゴール設定

- 1週目：登録と応募10件
- 2週目：初受注（単価より「完遂」優先）
- 3〜4週目：評価をもらい、次の案件へ

在宅ワークは「危険な案件を避ける目」さえ持てば、未経験でも安全に始められます。
