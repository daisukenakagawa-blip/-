# AIで仕事を10倍速にする実践テンプレート集【コピペOK・全部入り】

「AIを使えば仕事が速くなるのは分かるけど、具体的に何を打てばいいの?」——

このnoteは、抽象的な「AI活用のススメ」ではなく、**そのままコピペで使える
業務別プロンプト集**です。メール・議事録・資料・企画・分析まで、
明日から使えるテンプレートだけを詰め込みました。

## このnoteで分かること

- 成果が変わる「プロンプトの黄金型」
- 業務別コピペテンプレ（メール/議事録/資料/企画/Excel）
- AIに任せてはいけない仕事の線引き
- 出力品質を一段上げる追い打ちフレーズ

## まず覚える「プロンプトの黄金型」

良い指示は、次の4要素が入っています。

```
①役割：あなたは〇〇の専門家です
②前提：背景・読者・目的を伝える
③依頼：具体的に何を出してほしいか
④制約：文字数・トーン・形式（箇条書き等）
```

この型に沿うだけで、出力の質は大きく変わります。

## 1. メール作成テンプレ

```
あなたはビジネスメールのプロです。
以下の要点を、丁寧かつ簡潔なビジネスメールにしてください。
・宛先：取引先
・要点：納期を3日延ばしたいお願い／理由は◯◯
・トーン：申し訳なさを示しつつ前向きに
・200字程度、件名も提案して
```

## 2. 議事録の要約テンプレ

```
以下の会議メモを、次の形式で議事録にまとめてください。
【決定事項】【ToDo（担当者・期限つき）】【保留事項】
冗長な発言は削り、事実ベースで簡潔に。
---
（ここにメモを貼り付け）
```

## 3. 資料の骨子づくりテンプレ

```
あなたは資料構成の専門家です。
「◯◯（テーマ）」について、経営層向けに10枚のスライド構成を作ってください。
各スライドは「タイトル＋伝えるべき要点3つ」の形式で。
```

## 4. 企画の壁打ちテンプレ

```
私の企画案に、賛成派と反対派の両方の視点で指摘をしてください。
最後に、改善案を3つ提示してください。
企画案：（ここに貼る）
```

## 5. Excel/表計算テンプレ

```
次の操作をするExcelの数式を教えてください。
やりたいこと：（例）A列の日付から「曜日」をB列に表示したい
関数名だけでなく、セルに入れる完成形の数式で答えて。
```

## 6. 出力を一段上げる「追い打ちフレーズ」

- 「もっと具体例を3つ足して」
- 「専門用語を使わず、中学生にも分かる言葉で」
- 「反対意見も入れて、フェアに」
- 「箇条書きを表形式に直して」

## 7. AIに任せてはいけない仕事

- **事実の最終確認が必要なもの**（数字・固有名詞・法令）→ 必ず人がチェック
- **機密情報・個人情報**の入力 → 社内ルールを厳守
- **最終的な意思決定** → AIは案出し、決めるのは人

AIは「黄金型 × 業務別テンプレ」で一気に戦力化します。
まずはメールと議事録のテンプレを、今日の業務で1回使ってみてください。
