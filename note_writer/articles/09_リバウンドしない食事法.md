# リバウンドしない食事の"仕組み"の作り方【根性に頼らない設計図】

「ダイエットしてもすぐ戻る」「我慢が続かない」——
原因は意志の弱さではなく、**続かない方法を選んでいる**ことにあります。

このnoteは、極端な制限や根性論ではなく、**自然と続く食習慣の仕組み**の
作り方をまとめたものです。リバウンドの正体を理解し、設計で解決します。

> ⚠️ 本noteは一般的な生活習慣の情報提供です。持病のある方や治療中の方、
> 体調に不安のある方は、医師・管理栄養士など専門家にご相談ください。

## なぜリバウンドするのか

極端な食事制限は、短期では体重が落ちても**続けられない**ため、
反動で元の食生活に戻りやすくなります。大事なのは「**戻っても問題ない、
普段から続けられる食べ方**」を見つけることです。

## 1. 続く食習慣の3原則

1. **禁止food を作らない**（「絶対食べない」は反動を生む。量と頻度で調整）
2. **引き算より足し算**（減らすより、たんぱく質・野菜・水を"足す"）
3. **記録するだけ**（食べたものをメモするだけで意識が変わる）

## 2. 1食の組み立てテンプレ

```
①たんぱく質（肉・魚・卵・大豆）を手のひら1枚分
②野菜・きのこ・海藻を両手いっぱい
③主食（ご飯・パン）は握りこぶし1個分を目安に
④よく噛んでゆっくり
```

「何g」と計算しなくても、**手のひら基準**なら外食でも続けられます。

## 3. 我慢しないための工夫

- 甘いものは「やめる」ではなく「**時間と量を決める**」（例：昼に少量）
- 買い置きを減らす（家にあると食べる → 環境を変える）
- 空腹で買い物に行かない（衝動買いを防ぐ）

## 4. 続けるための"記録"のやり方

- アプリでもメモでもOK。**完璧を目指さない**
- 体重は「毎日同じ条件」で測り、**1週間の平均**で見る
- 1日単位の増減で一喜一憂しない

## 5. 停滞期との付き合い方

- 体重が落ちにくい時期は誰にでも来る
- ここで極端な制限に走るとリバウンドの入口に
- **同じ習慣を淡々と続ける**ことが、結局いちばんの近道

## まとめ

リバウンドしない食事は「我慢」ではなく「続く仕組み」で作ります。
まずは"足し算（たんぱく質・野菜・水）"と"記録するだけ"から始めましょう。
