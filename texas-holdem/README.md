# ♠ テキサスホールデム（6人固定・ノーリミット）

スマホ1台を6人で回して遊ぶ「ホットシート方式」のテキサスホールデム。
カジノルールに近い実装で、サイドポット／オールイン／スプリットポット／キッカー判定に対応しています。

## 仕様
- **6人固定**：開始時に6人の名前を入力。途中参加/退出なし
- **座席 Seat1〜6 固定**、時計回りに進行
- **ディーラーボタン**は毎ハンド時計回りに移動、SB/BB も自動移動
- **初期チップ 10,000** ／ **SB 100・BB 200**
- 中央にコミュニティカード5枚、周囲に6座席を固定配置
- 各座席に **名前／残りチップ／現在のベット／FOLD／ALL-IN** を表示
- **手札のプライバシー管理**：自分のターンになったらタップで手札を表示、アクション後は自動で非表示。ショーダウン時のみ全員公開
- **脱落・優勝判定**：チップが0で脱落、最後の1人を優勝者として表示

## 実装している本格ルール
- ✅ サイドポット（複数オールインを正しく分配）
- ✅ オールイン（ショート/フルレイズ判定、ボード自動ラン)
- ✅ スプリットポット（同点分配・端数チップはボタン左隣から配分）
- ✅ キッカー判定（7枚から最良5枚を評価、フルキッカー比較）
- ✅ ヘッズアップ（残り2人）のブラインド/アクション順序

## 遊び方
1. `index.html` をスマホのブラウザで開く
2. 6人の名前を入れて「ゲーム開始」
3. 自分の番が来たら「手札を見る」→ フォールド／チェック・コール／レイズ／オールイン
4. ショーダウンで結果表示 →「次のハンドへ」

## アプリとして使う
ブラウザの「ホーム画面に追加」で全画面・オフライン起動（PWA対応）。

## ローカルで試す
```bash
cd texas-holdem
python3 -m http.server 8000   # http://localhost:8000
```

## 構成
- `index.html` — ゲーム本体（HTML/CSS/JS、依存なし）
- `manifest.json` / `service-worker.js` — PWA
- `icons/` — アプリアイコン

## テスト
役判定・キッカー比較・サイドポット計算は単体テスト済み（19ケース）。
